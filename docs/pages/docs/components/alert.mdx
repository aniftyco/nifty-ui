# Alert

When your user makes a mistake, we will let them know.

---

import { IconPrompt, IconRipple } from '@tabler/icons-react';

import { Alert, AlertDescription, AlertTitle } from 'nifty-ui';

#### Default

<Alert variant="default">
  <IconPrompt className="h-4 w-4" />
  <AlertTitle>Heads up!</AlertTitle>
  <AlertDescription>You can add components to your app using the cli.</AlertDescription>
</Alert>

## Usage

```tsx
<Alert variant="default">
  <IconPrompt className="h-4 w-4" />
  <AlertTitle>Heads up!</AlertTitle>
  <AlertDescription>You can add components to your app using the cli.</AlertDescription>
</Alert>
```
