## Props

### Size

Small

```tsx
<Checkbox size="sm" id="foo" />
```

Default

```tsx
<Checkbox size="default" id="foo" />
```

Large

```tsx
<Checkbox size="lg" id="foo" />
```

Extra Large

```tsx
<Checkbox size="xl" id="foo" />
```
