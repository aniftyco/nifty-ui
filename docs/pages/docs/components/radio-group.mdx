# Radio Group

A bunch of Radios, all tuned to a different station.

---

import { Label, RadioGroup, RadioGroupItem } from 'nifty-ui';

<div>
  <RadioGroup defaultValue="comfortable">
    <div className="flex items-center space-x-2">
      <RadioGroupItem value="default" id="r1" />
      <Label htmlFor="r1">Default</Label>
    </div>
  </RadioGroup>
</div>

## Usage

```tsx
<RadioGroup defaultValue="comfortable">
  <RadioGroupItem value="default" id="..." />
</RadioGroup>
```
