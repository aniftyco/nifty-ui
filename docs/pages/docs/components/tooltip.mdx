# Tooltip

Simple little Tooltip for your users, because they will need it.

---

import { Button } from 'nifty-ui';
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from 'nifty-ui';

<TooltipProvider>
  <Tooltip>
    <TooltipTrigger asChild>
      <Button variant="outline" className="px-2 rounded-full text-slate-50 font-semibold">
        Hover
      </Button>
    </TooltipTrigger>
    <TooltipContent>
      <p>So fancy, isn't it?</p>
    </TooltipContent>
  </Tooltip>
</TooltipProvider>

## Usage

```tsx
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger asChild>
      <Button variant="outline">...</Button>
    </TooltipTrigger>
    <TooltipContent>...</TooltipContent>
  </Tooltip>
</TooltipProvider>
```
